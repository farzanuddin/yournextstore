# Your Next Store

A simple e-commerce web app built with Next.js, displaying products from the [Fake Store API](https://fakeapi.platzi.com/).

## Pages

- **Home** — Featured products grid
- **Product Detail** — Full product view with image gallery and add-to-cart
- **Cart** — Slide-out cart with quantity controls and checkout

## Setup

```bash
bun install
cp .env.example .env.local
bun dev
```

Open [http://localhost:3000](http://localhost:3000).
